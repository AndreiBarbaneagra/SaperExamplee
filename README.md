# SaperExamplee

## Описание задачи

Этот проект представляет собой реализацию игры "Сапёр" (Minesweeper) на языке C++. Игра создана в рамках обучения программированию и демонстрирует применение основных концепций объектно-ориентированного программирования.

## Правила игры

Цель игры "Сапёр" - открыть все клетки, не содержащие мины, не открыв ни одной из клеток с миной. Игрок может открывать клетки, вводя в консоль координаты поля.

## Зависимости проекта

- C++ компилятор (рекомендуется поддержка стандарта C++11 и выше)
- CMake (для сборки проекта)

## Сборка приложения

1. Убедитесь, что у вас установлен C++ компилятор и CMake.
2. Клонируйте репозиторий: `git clone https://github.com/AndreiBarbaneagra/SaperExamplee.git`
3. Перейдите в каталог проекта: `cd SaperExamplee`
4. Создайте каталог для сборки: `mkdir build && cd build`
5. Сгенерируйте проект CMake: `cmake ..`
6. Соберите приложение: `cmake --build .`

После выполнения этих шагов у вас должно появиться исполняемое файл `game`, которое можно запустить для игры в "Сапёр".
